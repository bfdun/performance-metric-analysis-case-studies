# Overview
This repository contains a series of applied performance analysis case studies examining various performance metrics (sprint times, single-leg balance times, etc.). Each case study focuses on appropriate statistical analysis of performance metrics within the exercise science field. Output for each case study include interactive html files or presentation pdfs.

# Data Sources

Each case study includes performance test data that was either simulated or collected but de-identified.
